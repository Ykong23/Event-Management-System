# Event-Management-System
Designed and implemented event &amp; feedback modules. Modernized UI, added dynamic questionnaires, and created in-place editing for streamlined event management.

## Overview
The **Event Management System** is a customized solution developed for a client of **Apploqic Technologies PLT**. This project aims to streamline the management of events through an intuitive platform, allowing users to create, manage, and participate in events effectively. Built using **WordPress** and hosted on **CPanel**, the system integrates various functionalities to enhance user experience.

## Purpose
The primary goal of this project is to provide a comprehensive event management tool that simplifies the process of organizing events while incorporating user feedback through questionnaires. This feature allows event organizers to gather valuable insights from participants, enhancing the overall quality of future events.

## Features
* **User-Friendly Interface**: The system is designed with a focus on usability, enabling users to navigate easily and manage their events without technical expertise.
* **Questionnaire Section**: Users can create and customize their questionnaire questions related to the event. This feature allows for tailored feedback collection, helping organizers understand participant experiences and preferences.
* **Payment Integration**: The questionnaire is seamlessly integrated into the payment section, allowing users to fill out the survey before completing their registration. This ensures that feedback is collected at a crucial moment in the user journey.
* **Event Management Tools**: Users can create, edit, and manage event details, including dates, locations, and participant lists.

## Technology Stack
**Programming Language**: JavaScript, PHP, Bootstrap, HTML
**Platform**: WordPress
**Hosting**: CPanel

## Conclusion
The Event Management System successfully meets the needs of the client by providing a robust platform for managing events and gathering participant feedback. By incorporating the questionnaire into the payment section, we have enhanced user engagement and provided valuable insights for future event planning.
